## 用 Hexo 搭建了个人 blog

### Hexo 基本操作

- 安装

    ```npm install -g hexo-cli```

- 初始化hexo

    ```hexo init```

    - 当前目录必须是空的

- 新建文章

    ```hexo new [layout] name```

    - layout 可以是 post, page, draft 之一

    - 分别保存至 source/_posts, source, source/_drafts 中

    - 新建文件名可以在 _config.yml 里面指定, `new_post_name`, 默认为 `name.md`

- 临时开启 server

    ```hexo server```
